Function contains()


contains(string1, string2) returns true if string1 contains string2, otherwise it returns false.

Example:

elem[contains(@attr, 'value')] - all elements elem that have the attribute @attr containing the value value


Use contains(), to select the input element that has the attribute id containing the value example in the current context.
