Predicates


[...] is the syntax for a predicate. It is used to restrict a node-set to select only those nodes for which some condition, an expression converted to boolean, is true.

This expression may be a node, an attribute, a number, a function, a comparison, a boolean expression.

Child elements


You may use a node as condition for the predicate to specify that the element that you want to select has that node as a child.

Example:

elem[child1] - all elements elem that contain at least one child element child1

elem[child1][child2] - all elements elem that contain at least one child element child1 and at least one child element child2
Note: There are two predicates here. Each predicate is filtered by the next.


Use [....] to select all the div elements that contain a child element label and a child element input in the current context.
