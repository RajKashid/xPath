Attributes


@ is the syntax used for referring to attributes.
@attr - refers to the attribute 'attr'

You may use an attribute as condition for the predicate to specify that the element that you want to select has an attribute with a specific value.

Example:

elem[@attr = 'value'] - all elements elem that have the attribute @attr equal to value


Use [....], @, = to select the input element that has the attribute id equal to 'exampleName' in the current context.
