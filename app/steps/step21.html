Positions


Positions are relative to what you have selected. elem[n] selects the n'th element elem in the current context.

Example:

div/elem[2] - the second element elem from under the div elements in the current context. All the selected elements will be on the same level: first level under div.

div//elem[2] - the second element elem from any level under the div elements in the current context. All the selected elements may be on multiple levels under div.


Use [n] to select the 4th td from the 2nd tr under the tbody element in the table table, in the current context
