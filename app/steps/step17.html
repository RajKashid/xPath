Operator or


or operator is used when from two conditions at least one should be filled

Example:

elem[child1 or child1] - all elements elem that have at least either a child node child1 or a child node child2


Use or operator to select in the current context the input element that:
- is under div parent with id equal to 'forName'
- has the attribute type equal to 'text'
- has the attribute class containing either value 'first' or the value 'last'.
